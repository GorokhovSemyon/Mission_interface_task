# Реализация интерфейса абстрактных миссий

## О проекте

Проект представляет собой реализацию интерфейса абстрактных миссий.

## Поддерживаемые опции

* Типы миссий
    + Переместиться
    + Погрузиться
    + Всплыть
    + Возврат
* Действия с миссиями
    + Добавление
    + Редактирование
    + Перемешивание
    + Удаление
* Печать текущего списка миссий
* Действия с файлом
    + Чтение
    + Запись
